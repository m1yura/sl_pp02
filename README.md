# PP02 — Производственная практика (Маркетинг и реклама)

## Описание
Данный репозиторий содержит материалы производственной практики ПП.02 по направлению **«Маркетинг и реклама»**. В ходе практики выполнялись задачи по проектированию информационной системы для автоматизации деятельности рекламного агентства, включая учет клиентов, рекламных кампаний, креативов, медиапланов и финансовых показателей.

Практика выполнялась с использованием средств моделирования UML, систем управления базами данных и разработки программных интерфейсов.

---

## Содержание репозитория

### `docs/`
Документация по практике:
- Отчёт по производственной практике
- Описание предметной области
- Техническое задание
- UML-диаграммы (Use Case, Class, Sequence, Activity, State, Component)

### `scripts/`
Скрипты для работы с информационной системой:
- `create_db.sql` — создание базы данных рекламного агентства
- `add_client.sql` — добавление нового клиента-рекламодателя
- `create_campaign.sql` — создание рекламной кампании
- `add_creative.sql` — добавление креатива к кампании
- `create_mediaplan.sql` — формирование медиаплана
- `add_kpi_data.sql` — внесение показателей эффективности
- `create_invoice.sql` — выставление счета клиенту
- `close_campaign.sql` — завершение кампании и расчет ROI
- `get_campaign_report.sql` — формирование отчета по кампании
- `search_creatives.sql` — поиск креативов по параметрам

---

## Цель работы
Целью производственной практики являлось получение практических навыков анализа предметной области, проектирования информационных систем, автоматизации учета рекламных проектов, а также разработки структуры базы данных и программных компонентов для рекламного агентства.

---

## Используемые технологии
- Моделирование: UML (PlantUML / Draw.io)
- Базы данных: SQL (MySQL / PostgreSQL)
- Документирование: Markdown, GitHub
- Проектирование: ER-диаграммы, логическое моделирование
